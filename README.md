# permadeath
Think you're a good coder? Only the paranoid survive here.

## Installation

```
npm install permadeath
```

## Usage

### CLI
```
./permadeath (--dry-run) someFile.js
```

### Module

```
var permadeath = require('permadeath');
permadeath('someFile.js');
```
